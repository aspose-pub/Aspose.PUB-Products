<p>
MS Publisher es un programa para crear documentos que están enfocados a trabajar con información de diseño de página. El programa abre los siguientes formatos de archivo MS Publisher 2.0 o posterior, archivos con extensión .txt, archivos RTF, archivos MS Word, archivos HTML, texto Unicode y muchos más. Este software le permite crear sus propios documentos diseñados o usar los diseños formateados para su documento.
MS Publisher guarda archivos con la extensión .pub y no muchas aplicaciones aparte de Publisher pueden procesar dichos archivos. 
</p>
<p>
 Aspose.PUB para C++ es una biblioteca de procesamiento de formatos PUb de Microsoft. Permite a los desarrolladores leer y cargar archivos PUB, manipular metadatos y convertir el formato de archivo de publicación electrónica (.pub) a PDF. Ofrece una conversión rápida y confiable de PUB a PDF, HTML, XLSX, DOC y los formatos de imagen más populares. La funcionalidad también se implementa en aplicaciones multiplataforma gratuitas para que pueda probarla usted mismo ahora mismo.
</p>
