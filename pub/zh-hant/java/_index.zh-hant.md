<p>
MS Publisher 是一個用於創建專注於處理頁面佈局信息的文檔的程序。該程序打開下一個文件格式 MS Publisher 2.0 或更高版本、擴展名為 .txt 的文件、RTF 文件、MS Word 文件、HTML 文件、Unicode 文本等等。該軟件允許創建您自己設計的文檔或為您的文檔使用格式化的設計。
MS Publisher 以 .pub 擴展名保存文件，除了 Publisher 之外，沒有多少應用程序可以處理此類文件。

<p>
 Aspose.PUB for Java 是一個靈活且易於使用的庫來處理電子出版物文件。它支持將流行的電子出版物文件格式 (.pub) 加載和轉換為 PDF。它提供 PUB 到 PDF、HTML、XLSX、DOC 和最流行的圖像格式的快速可靠的轉換。該功能也在免費的跨平台應用程序中實現，因此您現在可以自己嘗試。
</p>
