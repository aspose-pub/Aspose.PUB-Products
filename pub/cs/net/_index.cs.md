<p>
MS Publisher je program pro vytváření dokumentů zaměřených na práci s informacemi o rozvržení stránky. Program otevře další formáty souborů MS Publisher 2.0 nebo novější, soubory s příponou .txt, soubory RTF, soubory MS Word, soubory HTML, text Unicode a mnoho dalších. Tento software umožňuje vytvářet vlastní navržené dokumenty nebo používat naformátované návrhy pro váš dokument.
MS Publisher ukládá soubory s příponou .pub a ne tolik aplikací kromě Publisheru takové soubory dokáže zpracovat. 
</p>
<p>
Aspose.PUB for .NET je flexibilní a snadno použitelná knihovna pro práci se soubory elektronických publikací. Podporuje načítání a převod oblíbeného formátu souborů elektronické publikace (.pub) do PDF. Nabízí rychlý a spolehlivý převod PUB do PDF, HTML, XLSX, DOC a nejoblíbenějších obrazových formátů. Funkce je také implementována v bezplatných multiplatformních aplikacích, takže si ji můžete vyzkoušet sami hned teď.
</p>
