<p>
MS Publisher é um programa para criar documentos focados para trabalhar com informações de layout de página. O programa abre os próximos formatos de arquivo MS Publisher 2.0 ou posterior, arquivos com extensão .txt, arquivos RTF, arquivos MS Word, arquivos HTML, texto Unicode e muito mais. Este software permite criar seus próprios documentos projetados ou usar os projetos formatados para o seu documento.
O MS Publisher salva arquivos com extensão .pub e não muitos aplicativos além do Publisher podem processar esses arquivos. 
</p>
<p>
 Aspose.PUB para Java é uma biblioteca flexível e fácil de usar para trabalhar com arquivos de publicação eletrônica. Ele suporta o carregamento e conversão de formato de arquivo de publicação eletrônica popular (.pub) para PDF. Oferece conversão rápida e confiável de PUB para PDF, HTML, XLSX, DOC e os formatos de imagem mais populares. A funcionalidade também é implementada em aplicativos gratuitos de plataforma cruzada para que você possa experimentá-la agora mesmo.
</p>
